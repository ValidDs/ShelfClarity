# ShelfClarity

Working brand for a beauty shelf audit and routine planning product.

**Core promise:** Your beauty shelf, audited. Understand what to keep, pause, replace, and buy next.

## Current status

This repo contains the first visual landing-page concept and product planning documents for validation.

## Important caveat

ShelfClarity is a working name. Preliminary availability checks were cleaner than other options, but this is not legal clearance. Before locking the name, run proper trademark, domain, and app-store review.

## Safety positioning

ShelfClarity provides educational beauty routine organization. It does not diagnose, treat, cure, prevent, or replace advice from a licensed professional.
